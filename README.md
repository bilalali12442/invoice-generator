## 📖 کیسے کام کریں | How to Use

### **Step 1: Party Name درج کریں | Enter Party Name**
```
سب سے اوپر "PARTY NAME" میں اپنے کسٹمر کا نام لکھیں
Write customer name in "PARTY NAME" field
```

### **Step 2: Items شامل کریں | Add Items**
```
"+ Item Add" دبائیں
Click "+ Item Add" button
```

### **Step 3: Item کی تفصیلات | Fill Item Details**
```
Item کا نام (Item Name)
Rate (نرخ)
Quantity (مقدار)
→ Total خودکار طور پر calculate ہوگی
```

### **Step 4: Items Save کریں (Optional) | Save Items**
```
"📦 Items" دبا کر اپنے پسندیدہ items save کریں
اگلی بار جلدی select کر سکتے ہو
```

### **Step 5: Balance شامل کریں (Optional) | Add Balance**
```
Previous Balance - پہلے سے جو رقم بقایا ہے
Received - کتنی رقم ملی ہے
BALANCE - خودکار طور پر calculate ہوگی
```

### **Step 6: Save کریں | Save Invoice**
```
"💾 Save" دبا کر invoice محفوظ کریں
"📋 History" میں دیکھ سکتے ہیں
```
